A Rust client for ClickhouseDB

# Setup
## Clickhouse DB
1. Install Clickhouse: 
```
curl https://clickhouse.com/ | sh
```
2. Start the server:
```
./clickhouse server
```

# Rust Client
1. Run `cargo build` to install dependencies and build binaries.
2. --
